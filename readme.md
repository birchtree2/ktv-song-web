# ktv-song-web

KTV Song List Web

前端支持自动解析b站分享字符串

使用方法(确保安装了yarn/npm/pnpm等和Node.js)

```shell
git clone https://github.com/StarFreedomX/ktv-song-web.git

cd ktv-song-web

yarn install
# npm install
# pnpm install

# 复制一份 .env.example 到 .env
# 然后在 .env 中修改你想修改的配置

# 启动
yarn start
# npm run start
# pnpm start
```
